# sleepybee
대학에서 물리학을 전공하였고 만들어 보고 싶은 서비스가 있어서 2016년 안드로이드 개발을 시작했습니다.
스티브 잡스의 connecting the dots라는 말을 좋아합니다. 프로그래머가 되어야 겠다고 생각본 적이 없었는데 11살 때부터 html을 만지고 RPG 게임이나 미궁 게임을 만들던 것, 대학 때 C언어를 수강한 것들이 각각의 dots가 되어 이어져서 지금에 이르른 것 같습니다. 이후의 삶도 흥미있는 것을 따라가다보면 또 다른 데에 도달할 지도 모르겠다고 기대하고 있습니다.
개인 개발용 닉네임으로 쓰고 있는 sleepybee(졸린벌)은 본명인 leeseulbee의 애너그램을 짜다 만들었습니다.
